**Git e Git Hub** 

Commit > Tree > Blobs = Sistema distribuido seguro.
<u>Tracked:</u>
Unmodified > Modified = Editar um arquivo.
Modified > Staged = Arquivo em staged.
Unmodified > Modified > Staged = Adicionar um arquivo.
Staged > Modified > Unmodified = Commit.
Unmodified > Untracked = Remover arquivo.
<u>Comandos para repositório (Git bash here na pasta):</u>
ls > git status > git add . > git status > git commit -m "adicionar..." > git push origin main.
